# Red Batiz Firebase
An implementation of [RedBatiz](https://github.com/Angel45604/redBatizAngular) for Android, made in native android following the MVP architecture.
It features the Firestore Database and Storage Bucket from Firebase for post data and image storage. It also manages authentication with e-mail addressess and Facebook accounts.
